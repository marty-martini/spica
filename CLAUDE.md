# Spica

spica（おとめ座α星・真珠星）。
紀元前2世紀、ヒッパルコスはスピカの観測記録を約150年前の先人の記録と突き合わせ、そのわずかなズレから歳差運動を発見した。
「記録を蓄積し、時を越えて比較することで、目に見えない変化を捉える」——本プロダクトの思想そのもの。
名の意味は「麦の穂」。
測定ウィンドウという収穫の季節に実りをもたらす、蒼羽が紡ぐ星々の一番星。
